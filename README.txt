# Proyecto Dimension

Este proyecto es una aplicación web que utiliza un backend en Node.js y una base de datos MariaDB, junto con herramientas de monitoreo como Prometheus y Grafana. La interfaz de usuario está construida utilizando HTML5 y CSS, y es completamente responsiva.

## Requisitos

- Docker
- Docker Compose

## Instalación

1. Clona el repositorio:

   bash
   https://github.com/XxIMDARIOxX/raul_final
   cd raul_final

2. Asegúrate de tener Docker y Docker Compose instalados en tu máquina.

3. Ejecuta el comando para levantar el contenedor:

	bash
	docker-compose up -d

## Uso

	La aplicación web se puede acceder a través de la URL `http://localhost:3000`.

	Grafana se puede acceder en `http://localhost:3000` (usuario: admin, contraseña: admin).

	Prometheus se puede acceder en `http://localhost:9090`.
